# 15 个怪物 ChatGPT 图片生成 Prompt

把这 15 个 prompt 一个一个丢给 ChatGPT (gpt-4o image gen)，每张生成后保存到这个文件夹，文件名按下面表格。

如果完全没生图也行 — App 会自动回退到 emoji。

---

## 公共风格指令（每次复制到 prompt 末尾，保持一致风格）

```
Cute friendly cartoon style, kawaii Japanese mascot aesthetic, vibrant flat colors,
thick black outlines, simple solid background (single warm color), no text,
512x512 square, child-friendly, for a 5th grade spelling game.
```

## 怎么用

1. 一次性方法：复制全部 15 个 prompt + 公共风格指令，丢给 ChatGPT 让它批量生成
2. 单个方法：每次复制一个 prompt + 公共风格，让 ChatGPT 出图，下载保存到这个文件夹
3. 文件名必须**完全匹配上表**（slime.png, bat.png 等小写）
4. PNG 透明背景最好，纯色背景也行（App 会显示背景色）

不生图也没关系，App 默认用 emoji，照样能玩。
